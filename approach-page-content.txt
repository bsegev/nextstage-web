DESIGN LANGUAGE ANALYSIS FROM 5 COMPONENTS:
===========================================

## 1. Hero.tsx - Premium Foundation
**Key Patterns**:
- `font-display` for headlines with `tracking-tight` and `leading-[0.9]`
- Gradient text: `bg-gradient-to-r from-obsidian via-accent to-obsidian bg-clip-text text-transparent`
- Motion: `initial={{ opacity: 0, y: 20 }}` with staggered delays
- Responsive scaling: `text-4xl sm:text-3xl md:text-4xl lg:text-5xl xl:text-6xl`
- Subtle animations: floating orbs, grid patterns, gradient animations
- Color palette: obsidian, bone, accent (#FFE0D7)

## 2. CTASection.tsx - Dynamic Storytelling
**Key Patterns**:
- Parallax backgrounds with `useTransform(scrollYProgress)`
- Animated SVG paths with flowing gradients
- Counter animations with eased transitions
- Complex layered backgrounds: gradients + SVG + motion
- Stats rotation with pause on hover
- Mobile-first responsive design

## 3. SpeedProof.tsx - Comparison Excellence
**Key Patterns**:
- Side-by-side comparison layout (desktop) vs stacked (mobile)
- Color coding: grey for traditional, accent for NextStage
- Timeline visualization with animated progress bars
- Highlight badges for key differentiators
- Staggered reveal animations with `whileInView`
- Clean typography hierarchy

## 4. BentoGrid.tsx - Modular System
**Key Patterns**:
- Grid-based layout with `md:auto-rows-[18rem] md:grid-cols-3`
- Hover effects: `group-hover/bento:translate-x-2`
- Consistent spacing and border radius
- Shadow system for depth
- Flexible content areas

## 5. AboutCTA.tsx - Sophisticated Interaction
**Key Patterns**:
- Complex geometric animations with `useTransform`
- Availability indicators with pulsing animations
- Convergence metaphors through SVG geometry
- Layered background systems
- Mobile-optimized versions of desktop elements

## DESIGN PRINCIPLES IDENTIFIED:
===============================
1. **Motion with Purpose**: Every animation serves the narrative
2. **Responsive Excellence**: Mobile-first with desktop enhancements
3. **Gradient Mastery**: Sophisticated color transitions
4. **Typography Hierarchy**: Display fonts for impact, clean body text
5. **Layered Depth**: Multiple background layers create richness
6. **Geometric Sophistication**: SVG animations and mathematical precision
7. **Color Psychology**: Obsidian (authority), Bone (clarity), Accent (energy)
8. **Interaction Feedback**: Hover states and micro-interactions
